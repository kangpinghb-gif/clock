# 📞 我的代办 — 用户使用手册

## 这是什么？

一个飞书机器人。你给它发消息，到时间它打电话提醒你。

---

## 🚀 开始使用

### 第 1 步：找到机器人

打开飞书 → 搜索 **「我的代办」** → 进入对话

### 第 2 步：绑定手机号

**首次使用必须绑定手机号**，否则无法接电话。

发送：
```
/bind 138xxxx8000
```
（把号码换成你自己的）

### 第 3 步：创建待办

| 示例 | 说明 |
|------|------|
| `明天下午3点提醒我对账` | 最常见用法 |
| `5分钟后提醒我开会` | 相对时间 |
| `今晚8点吃药` | 当天时间 |
| 🎤 发语音消息 | 自动转文字 |

### 第 4 步：收到确认

```
✅ 已记录！
📝 内容：明天下午3点提醒我对账
⏰ 时间：2026-06-16 15:00
```

### 第 5 步：到时间接电话 📞

手机响起 → 接听后播放：

> *"您好，您有一条待办提醒：明天下午3点提醒您对账，请及时处理"*

---

## 📋 命令

| 命令 | 功能 | 必须 |
|------|------|------|
| `/bind 138xxxx8000` | 绑定你的手机号 | ✅ **必做** |
| `/list` | 查看你的待办列表 | ❌ 可选 |
| `帮助` | 查看使用说明 | ❌ 可选 |

---

## ⏰ 时间写法

| 输入 | 理解的时间 |
|------|-----------|
| `5分钟后提醒我开会` | 当前时间 + 5分钟 |
| `3小时后提醒我下班` | 当前时间 + 3小时 |
| `明天下午3点提醒我对账` | 次日 15:00 |
| `今晚8点吃药` | 当天 20:00 |
| `后天早上9点出发` | 后天 09:00 |
| `下周一上午10点开会` | 下周一 10:00 |

---

## ❓ 常见问题

### 电话没接到怎么办？
系统会自动重试最多 3 次，每次间隔 5 分钟。

### 语音识别错了怎么办？
- 发语音时放慢语速
- 或者直接发**文字**更准确

### 费用要我出吗？
电话费由系统承担，你接听**完全免费**。

### 可以取消待办吗？
当前支持查看（`/list`），取消功能开发中。

### 搜不到机器人？
联系管理员，需要先在飞书后台设置你的可用范围。
