# 📞 我的代办 — 电话提醒系统

通过飞书机器人 **「我的代办」** 发送文字或语音消息，系统自动记录待办事项，到时间后拨打你的手机电话提醒。

---

## 🚀 快速开始

### 1. 找到机器人

打开飞书 → 搜索 **「我的代办」** → 进入对话

### 2. 绑定手机号

**首次使用必须先绑定手机号**，否则无法接听电话提醒：

```
/bind 138xxxx8000
```

> 把 `138xxxx8000` 换成你自己的手机号码

### 3. 创建待办

发文字或语音消息给机器人：

| 示例 | 说明 |
|------|------|
| `明天下午3点提醒我对账` | 最常见用法 |
| `5分钟后提醒我开会` | 相对时间 |
| `今晚8点吃药` | 当天时间 |
| 🎤 发语音消息 | 自动识别成文字 |

### 4. 收到确认

```
✅ 已记录！
📝 内容：明天下午3点提醒我对账
⏰ 时间：2026-06-16 15:00
```

### 5. 到时间自动接电话 📞

到了设定时间，手机收到来电：

> **来电号码：** 阿里云语音专线
> **接听后播放：** "您好，您有一条待办提醒：明天下午3点提醒您对账，请及时处理"

---

## 📋 支持的命令

| 命令 | 功能 |
|------|------|
| `/bind 138xxxx8000` | **必做！** 绑定你的手机号（不绑定无法接电话） |
| `/list` | 查看你的待办列表 |
| `帮助` | 查看使用说明 |

---

## 🎤 语音输入

支持直接发语音消息，系统会自动转为文字并创建待办。

---

## ⏰ 时间格式

| 你说的话 | 系统理解的时间 |
|----------|---------------|
| `5分钟后` | 当前时间 + 5分钟 |
| `3小时后` | 当前时间 + 3小时 |
| `明天下午3点` | 次日 15:00 |
| `今晚8点` | 当天 20:00 |
| `后天早上9点` | 后天 09:00 |
| `下周一上午10点` | 下周一 10:00 |

---

## 👥 多用户使用

其他用户直接搜 **「我的代办」** 即可使用，各自绑定自己的手机号：

- 用户A → `/bind 138xxxxxxx` → 创建待办 → 打给用户A
- 用户B → `/bind 139xxxxxxx` → 创建待办 → 打给用户B

> **注意：** 如果搜不到机器人，请联系管理员在飞书控制台设置「可用范围」

---

## ❓ 常见问题

### 电话没接到会重试吗？
会。最多重试 3 次，每次间隔 5 分钟。

### 费用多少？
阿里云语音通知约 0.05~0.1 元/分钟（接通才计费），个人使用几乎可以忽略。

### 语音识别不准怎么办？
- 发语音时语速放慢、发音清晰
- 也可以直接用**文字输入**

### 可以取消待办吗？
目前支持查看待办（`/list`），取消功能正在开发中。

### 服务异常怎么办？
联系管理员查看日志：
```bash
journalctl -u todo-recv.service -f
journalctl -u todo-remind.service -f
```
