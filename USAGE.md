# 📞 电话提醒系统 — 用户使用说明

## 这是什么？

一个飞书机器人。你给它发文字或语音，到时间它**自动打电话**提醒你。

---

## 🚀 快速上手

### 第 1 步：找到机器人

打开飞书 → 搜索 **「我的代办」** → 开始对话

### 第 2 步：绑定手机号（首次必做）

发送：
```
/bind 138xxxx8000
```
（换成你自己的手机号）

### 第 3 步：创建待办

| 示例 | 说明 |
|------|------|
| `明天下午3点提醒我对账` | 最常用 |
| `5分钟后提醒我开会` | 相对时间 |
| `今晚8点吃药` | 当天时间 |
| 🎤 发语音消息 | 自动识别 |

### 第 4 步：到时间接电话 📞

手机响起，接听后播放：
> "您好，您有一条待办提醒：明天下午3点提醒您对账"

---

## 📋 常用命令

| 命令 | 作用 |
|------|------|
| `/bind 138xxxx8000` | **绑定手机号（必做）** |
| `/unbind` | 解绑手机，不再接电话 |
| `/list` | 查看待办列表 |
| `/cancel 3` | 撤销编号为 3 的待办 |
| `/cancel all` | 撤销所有待办 |
| `帮助` | 查看使用说明 |

---

## 📞 打给指定号码

```
给138xxxx8000打电话提醒他去开会
```

到时间会打给 **138xxxx8000**，而不是你自己。

如果不指定号码，默认打给你绑定的手机。

---

## ⏰ 支持的时间写法

| 写法 | 理解成 |
|------|--------|
| `5分钟后` | 当前 + 5分钟 |
| `3小时后` | 当前 + 3小时 |
| `明天下午3点` | 次日 15:00 |
| `今晚8点` | 当天 20:00 |
| `后天早上9点` | 后天 09:00 |
| `下周一上午10点` | 下周一 10:00 |
| `12:20` | 当天 12:20 |

---

## ❓ 常见问题

**电话没接到会重试吗？**
会，最多重试 3 次，每次隔 5 分钟。

**语音识别错了怎么办？**
可以改发文字，更准确。

**费用要我出吗？**
不用，接听完全免费。

**搜不到机器人？**
联系管理员，在飞书后台设置可用范围。
